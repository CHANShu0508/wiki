# 基础知识

## 数字与操作符

```python
3 + 2   # Output is 5
3 * 5   # Output is 15
3 / 2   # Ooutput is 1.5
3 // 2  # Output is 1
-3 // 2 # Output is -2
3 ** 2  # Output is 9
```

::: theorem Floor division
The `//` operator is called [floor division](https://docs.python.org/3/glossary.html#term-floor-division), which is the mathematical division that rounds down to nearest integer
:::
